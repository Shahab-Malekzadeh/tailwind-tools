Aspect Ratio

Utilities for controlling the aspect ratio of an element. more

Class Properties
aspect-auto aspect-ratio: auto;
aspect-square aspect-ratio: 1 / 1;
aspect-video aspect-ratio: 16 / 9;

Setting the aspect ratio

Use the aspect-{ratio} utilities to set the desired aspect ratio of an element. Resize the box to see the changes.
