Starting and ending lines

Use the "col-start-{n}" and "col-end-{n}" utilities to make an element start or end at the nth grid line. These can also be combined with the "col-span-{n}" utilities to span a specific number of columns. Note that CSS grid lines start at 1, not 0, so a full-width element in a 6-column grid would start at line 1 and end at line 7. more
