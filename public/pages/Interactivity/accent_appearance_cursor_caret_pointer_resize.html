Pointer Events

Utilities for controlling whether an element responds to pointer events. more

Controlling pointer event behavior

Use "pointer-events-auto" to revert to the default browser behavior for pointer events (like ":hover" and "click").
Use "pointer-events-none" to make an element ignore pointer events.
The pointer events will still trigger on child elements and pass-through to elements that are “beneath” the target.
